# Security Policy

This is a static single-page game — no authentication, no accounts, no PII, no
payments. The attack surface is the client bundle and the small Express static
server (`server.js`).

## Supported versions

Only the latest code on `main` is supported.

## Reporting a vulnerability

Please **do not open a public issue** for security problems. Instead, use the
repo's **Security tab → "Report a vulnerability"** (GitHub private vulnerability
reporting). You can expect an initial response within about **7 days**.
